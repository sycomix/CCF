  • method: Method name

  • f: Method implementation

  • rw: Flag if method will Read, Write, MayWrite

  • params_schema: JSON schema for params object in requests

  • result_schema: JSON schema for result object in responses

  • forwardable: Allow method to be forwarded to primary

Populate out with all supported methods

This is virtual since the default handler may do its own dispatch internally, so derived implementations must be able to populate the list with the supported methods however it constructs them.
